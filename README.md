# 顧客名簿データクレンジング（サンプル）

汚れた顧客名簿CSVを、Pythonで自動整形してExcelに出力するサンプルです。
手作業だと数時間かかる名簿整形を、スクリプト1本で数秒・ノーミスで再現します。

## できること（このスクリプトが直す「汚れ」）

| 項目 | Before（汚いデータ） | After（整形後） |
|---|---|---|
| 会社名の表記ゆれ | `(株)鈴木製作所` / `㈱田中工業` / `スター工業（株）` | `株式会社鈴木製作所` / `株式会社田中工業` / `スター工業株式会社` |
| 全角英数字 | `０９０－１２３４－５６７８` / `ＮＴＴ` | `090-1234-5678` / `NTT` |
| メールアドレス | `YAMADA@example.com` / `WATANABE@Example.Com` | `yamada@example.com` / `watanabe@example.com` |
| 電話番号の形式バラつき | `06 6789 1234` / `090.2222.3333` / `0312345678` | `06-6789-1234` / `090-2222-3333` / `03-1234-5678` |
| 郵便番号 | `〒100-0001` / `1500002` / `４６００００２` | `100-0001` / `150-0002` / `460-0002` |
| 都道府県の略記 | `東京` / `大阪` / `京都` | `東京都` / `大阪府` / `京都府` |
| 日付の表記ゆれ | `R6.3.1` / `令和6年5月20日` / `2024.4.15` | `2024-03-01` / `2024-05-20` / `2024-04-15` |
| 金額 | `¥1,200,000` / `800000円` / `２，２００，０００` | `1200000` / `800000` / `2200000` |
| 余分な空白 | `鈴木 花子 `（末尾空白）/ 全角スペース | `鈴木 花子` |
| 重複行 | 同じ顧客IDの行が2件 | 1件に統合（削除件数を報告） |

整形しても直らない「中身が空」の問題（担当者名・金額の欠落、解釈できない日付）は、
勝手に埋めずに**件数をレポートで報告**します（人の確認が必要な箇所を明示）。

## 実行結果（18行 → 17行）

```
顧客名簿 クレンジング結果レポート
====================================
- 入力行数: 18 行
- 担当者名が空欄の行: 1 件 (要確認)
- 売上金額が空欄/不明の行: 2 件 (要確認)
- 顧客ID重複により削除: 1 行 (ID: 1)
- 出力行数: 17 行
```

## ファイル構成

```
data-cleaning-sample/
├─ raw/顧客名簿_raw.csv       入力（汚いデータ）
├─ clean.py                   整形スクリプト
├─ output/
│  ├─ 顧客名簿_clean.xlsx      整形後データ
│  └─ cleaning_report.txt      変更内容レポート
└─ README.md                  このファイル
```

## 使い方

```bash
pip install pandas openpyxl
python clean.py
```

## 技術メモ

- 全角→半角の変換は `unicodedata.normalize("NFKC", ...)` を使用（記号・英数字をまとめて正規化）
- 日付は西暦・和暦（令和/R表記）の両方を正規表現で解釈し ISO形式 `YYYY-MM-DD` に統一
- 環境: Python 3 / pandas / openpyxl

---
*カスタマイズ可能です。列名・整形ルール・出力形式（CSV/Excel）はご希望に合わせて調整できます。*
